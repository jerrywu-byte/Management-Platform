TableCardStudio V3.1.9 架構版

使用方式：
1. 解壓縮整個資料夾。
2. 雙擊 start_windows.bat，或直接用 Chrome / Edge 開啟 index.html。
3. 左側可切換四個模組：
   - 桌卡 / 菜卡 / 位卡：沿用 V2.7.1
   - 桌圖設計：先建立架構，內容待建構
   - 尾款試算表：先建立單機架構，之後移植同事 GitHub 版本
   - 場次資訊確認書：先建立架構，內容待建構

注意：
- 這版不用 GitHub、不需要網路。
- V2 的桌卡、菜卡、位卡先保留在 modules/cards/index.html。
- 三個 V3 新功能先架構在同一平台，後續再逐步補內容。

V3.1 更新：
- 將桌卡模組的列印按鈕移到下方同一列。
- 新增圖片自動判斷 / OCR 接口，正式版可再接本機 OCR。
- 尾款試算表保留單機移植區，不依賴 GitHub。

V3.1.1 更新：
- 菜卡停車折抵文字、固定提醒、場次名稱字體調整為 12pt。
- QRcode 位置不動。
- 「菜卡藍框場次名稱預覽」改為「場次名稱」。
- 「紅框固定提醒文字，可改版」改為「固定提醒，可改版」。
- 主標題不再直接接版本，版本改放固定位置。

V3.1.2 更新：
- 菜卡停車折抵文字、固定提醒文字、場次名稱改為 10pt。
- QRcode 位置往右移 0.5mm。
- 儲存按鈕名稱改為「暫存目前畫面」、「另存場次檔」、「讀取場次檔」。
- 另存 JSON 檔名格式改為「場次檔-日期時段廳房宴會名稱.json」。
- 廳房名稱改為下拉式選單；紫艷系列與世紀盛典系列支援使用區域勾選後自動產生顯示名稱。
- 年份加入自動判斷：MMDD 為今天或未來判斷今年，已早於今天判斷明年；仍可手動修改。

V3.1.3 更新：
- 「讀取場次檔」按鈕高度與字型調整為與旁邊「另存場次檔」一致。
- 「4. 製作資料」改為「3. 製作資料」。
- 「QRcode 與菜卡固定文字」改為第 4 區，移到製作資料下方，並以橫式區塊呈現。
- 若年份自動判斷為下一年，提醒改成紅色小字，顯示在年份欄位下方。
- 尾款試算表已依附件檔整理進 V3 的「尾款試算表」模組，並改為單機本機使用，移除外部 logo 圖片依賴。

V3.1.4 更新：
- 外層「宴會場次製作平台」右側加入典華正式 logo 圖檔。
- 桌菜卡製作：停車折抵文字與固定提醒改為右側上下排列的橫式輸入區。
- 菜卡成品 QRcode 往右靠近文字，QRcode 與文字距離設為 5mm。
- 桌菜卡製作頁「廳房名稱」改為「廳別」。
- 尾款試算表取消內部 logo 顯示，改使用外層平台右側 logo。
- 尾款試算表「客戶／主題名稱」改為「客戶／宴會名稱」。
- 尾款試算表廳別套用桌菜卡同一套下拉式選單與合併廳別判斷。

V3.1.5 更新：
- 桌卡文字若單行字數過多，預覽與列印會自動縮小字體，避免超出版面。
- 桌卡解析支援無空白格式，例如「1-1男方親友」、「11男方親友」，會自動把前方桌號拆到第一行。
- 修正 start_windows.bat 讀不到 index.html 的問題，改用絕對路徑開啟。
- 外層「宴會場次製作平台」LOGO 改放在標題左側，版本保留在右側。

V3.1.6 更新：
- 桌卡解析支援英文桌號，例如 A、A桌、A男方親戚，解析原理與數字桌號相同。
- 場次資料支援無空白格式自動判斷，例如 0712午伊麗莎白姚林府喜宴Jimmy。
- 尾款試算表「其他加購」固定四個預設品項，清空重填不會清掉這四個品項名稱。
- 尾款試算表「說明文字」與「左下提醒」保留預設內容，清空重填不會清空。
- 其他費用與其他加購的招待欄改為下拉式選單：原價、10%、5%、招待。
- 已選 10%、5%、招待的項目不再列入服務費重複計算。
- 桌餐與服務輸入原價，右側即時顯示原價 +10%，試算表也以含 10% 的單價計算。

V3.1.6 菜卡微調：
- 菜名字體明確維持 13pt。
- 菜卡左右安全空間 / 邊界由 15mm 調整為 10mm。

V3.1.6 菜卡微調：
- 菜單超過 12 道菜時，系統會自動縮小菜名與細項行距，並啟用下方 30mm 優化空間，讓菜單盡量完整顯示。

V3.1.6 修正：
- 修正桌卡 renderTables() 重複宣告，避免桌卡預覽與列印失效。
- 已針對純數字、純英文、英數桌號、含文字桌號做邏輯測試。

V3.1.6 修正：
- 修正 renderTables() 重複宣告。
- 修正 A桌、1-1 等單獨桌號被拆成兩行的問題。

V3.1.6 補充修正：
- 單獨桌號 A桌 / B2桌 / 1桌 / 1-1 也會顯示在第 2 行，並放大為 180pt。

V3.1.7 更新：
- 尾款試算表取消獨立「服務費」欄位，所有加成改由各品項的「服務費/稅金」欄位控制。
- 其他加購固定四個品項名稱，原價、數量、服務費/稅金仍可修改，並可新增加購項目。
- 其他費用固定五個品項名稱：場租、舞台、麥克風、視訊進系統、音訊進系統；後方欄位可修改，並可新增其他費用。
- 試算資料內「加價」更名為「服務費/稅金」。

V3.1.8 更新：
- 說明文字與左下提醒改為固定底文，不可被清空；使用者可在固定文字後方追加內容。
- 桌餐與服務的「含10%」改為「服務費」下拉式選單，可選 10%、5%、0%、招待。
- 餐飲服務、其他費用、其他加購均以各列的服務費/稅金選項計算。

V3.1.9 更新：
- 尾款試算器刪除「廳別顯示名稱」欄位，廳別選擇直接綁定試算表標題。
- 「客戶／宴會名稱」改為半寬，旁邊放「訂金已支付」。
- 刪除「分館／館別」欄位；試算表標題中的「新莊典華」固定不可修改。
- 修正餐桌與服務選 0% 會跳回 10% 的顯示錯誤，0% 顯示與計算皆為 0%。
- 左下提醒改為單行：「*上述僅為試算之費用，不含當日另新增費用及酒水。」並移除不必要的 \n。
