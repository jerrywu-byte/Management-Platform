桌卡製作管理平台 V2.7.1.2

使用方式：
1. 解壓縮 ZIP。
2. 雙擊 start_windows.bat，或用 Chrome / Edge 開啟 index.html。
3. 先按「貼入測試範例」確認流程。
4. 列印時務必在印表機選擇實際大小 / 不縮放 / 無邊框 / 高品質。

已支援尺寸：
- 帳篷式桌卡：148mm x 210mm，折起 148mm x 105mm。
- 菜卡：88mm x 227mm，葷桌/素桌版型，上方預留 33mm。
- 名卡/位卡：130mm x 140mm，折起 130mm x 70mm。

V2.7.1 修正：
- 第 4 欄位「製作資料」改為固定在頁面內，不再跟著往下跑、遮住預覽按鈕。
- 菜卡輸出上方預留 33mm。
- 菜名主標改為分散對齊。
- 樓層 / 場地補充改為下拉式：空白、2樓、3樓、5樓、6樓。
- QRcode 固定為 20mm x 20mm，並調整左側位置，對齊下方固定文字區的起始位置。

已支援：
- 場次字串解析：0712 午 伊麗 姚林府喜宴 Jimmy
- 桌卡三行判斷：例 1-1 男方爸爸公司桌 -> 1-1 / 男方爸爸 / 公司桌
- 菜卡編號移除與括號細項縮小
- 5小時優惠 QRcode 與其他優惠 QRcode 覆蓋規則
- 菜卡固定文字可修改
- 北歐菜單、法蘿菜單先預留欄位
- 匯出/匯入 JSON

注意：
V2.7.1 是單機網頁版，圖片自動判斷/OCR 尚未接入正式模組；目前圖片可上傳留存，文字請先手動貼入對應欄位。


V2.7.1 修正：
1. 桌卡文字改為粗體。
2. 貼入測試範例後，輸出類型改為「全部預覽」，下方會同時顯示桌卡、菜卡與名卡範例。


V2.7.1 更新：桌卡文字改為微軟正黑體 72pt 粗體且全部同大小；菜名與括號細項改為 13pt 分散對齊；固定提醒文字上移並保留底部 55mm 空白；位卡公司/職稱 24pt、姓名 54pt。
